## CrusherSDK

Crusher sdk can be accessed by using the global `crusherSdk` which will be available to
all the custom code actions.

<ul className="apiList" style={{listStyle: "none"}}>
    <li>
        <a href="#">crusherSdk.$(selector)</a>
    </li>
    <li>
        <a href="#">crusherSdk.setCookies(cookies)</a>
    </li>
    <li>
        <a href="#">crusherSdk.reloadPage()</a>
    </li>
    <li>
        <a href="#">crusherSdk.sleep(timeout)</a>
    </li>
</ul>

### crusherSdk.$(selector)
<ul className="apiList" style={{listStyle: "none"}}>
    <li><pre style={{display: "inline", padding: "0.25em"}}>selector</pre> {`<string>`} A selector to query for. See working with selectors for more details.</li>
    <li>returns: {`<Promise<null|CrusherSDKElement>>`}</li>
</ul>

The method finds the element with the specified selector and returns a sdk reference to it. If no elements matched with given selector, returns null.

### crusherSdk.setCookies(cookies)
<ul className="apiList" style={{listStyle: "none"}}>
    <li>
        <pre style={{display: "inline", padding: "0.25em"}}>cookies</pre> {`<Array<Object>>`}<br/>
        <ul className="apiList" style={{listStyle: "none"}}>
            <li><pre style={{display: "inline", padding: "0.25em"}}>name</pre> {`<string>`}</li>
            <li><pre style={{display: "inline", padding: "0.25em"}}>value</pre> {`<string>`}</li>
            <li><pre style={{display: "inline", padding: "0.25em"}}>url</pre> {`<string>`}</li>
            <li><pre style={{display: "inline", padding: "0.25em"}}>domain</pre> {`<string>`}</li>
            <li><pre style={{display: "inline", padding: "0.25em"}}>path</pre> {`<string>`}</li>
            <li><pre style={{display: "inline", padding: "0.25em"}}>expires</pre> {`<number>`} Optional.</li>
            <li><pre style={{display: "inline", padding: "0.25em"}}>httpOnly</pre> {`<boolean>`}</li>
            <li><pre style={{display: "inline", padding: "0.25em"}}>secure</pre> {`<boolean>`}</li>
            <li><pre style={{display: "inline", padding: "0.25em"}}>sameSite</pre> {`<"Strict"|"Lax"|"None">`} Optional.</li>
        </ul>
    </li>
    <li>returns: {`<Promise<boolean>>`}</li>
</ul>

Sets the cookies in the browser.

### crusherSdk.reloadPage()
<ul className="apiList" style={{listStyle: "none"}}>
    <li>returns: {`<Promise<boolean>>`}</li>
</ul>

Reloads the current page.

### crusherSdk.sleep(timeout)
<ul className="apiList" style={{listStyle: "none"}}>
    <li><pre style={{display: "inline", padding: "0.25em"}}>timeout</pre> {`<number>`} In milliseconds.</li>
    <li>returns: {`<Promise<boolean>>`}</li>
</ul>

Pauses the test for `timeout` milliseconds.

<ul className="apiList" style={{listStyle: "none"}}>

<ul className="apiList" style={{listStyle: "none"}}>

### crusherSDKElement.fill(text)
<ul className="apiList" style={{listStyle: "none"}}>
    <li><pre style={{display: "inline", padding: "0.25em"}}>text</pre> {`<string>`}</li>
    <li>returns: {`<Promise<boolean>>`}</li>

### crusherSDKElement.type(text\[,options])
<ul className="apiList" style={{listStyle: "none"}}>
    <li><pre style={{display: "inline", padding: "0.25em"}}>text</pre> {`<string>`}</li>
    <li><pre style={{display: "inline", padding: "0.25em"}}>options</pre> {`<string>`}<br/>
        <ul className="apiList" style={{listStyle: "none"}}>
            <li><pre style={{display: "inline", padding: "0.25em"}}>delay</pre> {`<number>`} Time to wait between key presses in milliseconds. Defaults to 0</li>
        </ul>
    </li>
    <li>returns: {`<Promise<boolean>>`}</li>
</ul>

Types the current element with the text, one character at a time.
