Test resposively across different devices. 90% of users come from mobile, thus it becomes extremely critical to ensure HQ.

## Creating tests for mobile
Recording for mobile is supported is only for one viewport. Although after saving test, you can different configuration like iPhone6, Apple to run tests.

1.) **Open crusher recorder** and **enter URL** you want to test.

2.)   Click **top bar > device dropdown**. Choose **Mobile**.

![enter image description here](https://i.imgur.com/vcUEwOu.png)

3.) Create test and perform actions.

4.) Click on save test.
